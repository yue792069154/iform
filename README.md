# iForm
![](https://img.shields.io/badge/View%20UI-%20v4.0.3-blue)

```
iForm是一款基于iview、vant两套ui框架实现的动态拖拽表单开源工具。
```
- iForm于2020年03月16日正式启动研发，目前不具备上线条件。现招募广大前端开发爱好者共同研发。

### Install
```
npm install
```

### Development
```
npm run init
npm run dev
```

### Production(Build)
```
npm run build
```
